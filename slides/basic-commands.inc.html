Basic Commands

First-time setup

git config --global user.name "Your Name"
git config --global user.email "your.email@uwaterloo.ca"

Getting a repository

git clone https://github.com/waterloo-rocketry/repo-name.git

Downloads the entire project and its history

Checking status

git status

Shows which files have been modified, added, or deleted

The Git workflow

Working Directory
Your actual files
↓ git add
Staging Area
Files ready to commit
↓ git commit
Repository
Permanent history
