Branches Overview

What are branches?

Branches let you work on features independently without affecting the main code

Visual Example:
main: A---B---C---F---G
\
feature: D---E

A, B, C: Initial commits on main branch

D, E: Feature development on separate branch

F: Merge feature back into main

G: Continued development on main

Why use branches?

🛡️ Keep main stable
🚀 Work on features safely
👥 Collaborate without conflicts
🔄 Easy to switch contexts
