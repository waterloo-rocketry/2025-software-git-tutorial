GitHub Issues

Issues are how we track bugs, feature requests, and tasks in our projects.

Creating a good issue

  • Clear title - summarize the problem in one line
  • Detailed description - explain what's happening
  • Steps to reproduce - how can someone else see the problem?
  • Expected vs actual behavior - what should vs does happen
  • Environment details - OS, software versions, hardware

How to reproduce section

Steps to reproduce:
1. Clone the repository
2. Run `python sensor_test.py`
3. Connect the temperature sensor to pin A0
4. Observe the output in the terminal

Expected: Temperature readings in Celsius
Actual: Getting negative values and crashes
