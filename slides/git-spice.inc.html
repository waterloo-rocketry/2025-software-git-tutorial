Git Spice & Atomic PRs

What is git-spice?

Git-spice is a tool that helps you create "stacked" pull requests - multiple related PRs that build on each other.

Installation

# Install via Homebrew (macOS/Linux)
brew install git-spice

# Or download from GitHub releases
# github.com/abhinav/git-spice

Basic git-spice workflow

gs branch create feature-part1 # Create first branch
# Make changes and commit
gs branch submit # Submit first PR

gs up # Move to next branch
gs branch create feature-part2 # Create stacked branch
# Make changes and commit
gs branch submit # Submit stacked PR

gs stack submit # Update entire stack

This creates PRs where part2 builds on part1, with proper dependency tracking.

Key Commands

gs branch create - Create new branch

gs branch submit - Submit current branch as PR

gs stack submit - Submit entire stack

gs up/down - Navigate between branches

gs repo sync - Sync with trunk after merges

What are Atomic PRs?
